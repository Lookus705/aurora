# Salones y clínicas — MVP local

Base inicial para construir un asistente conversacional para salones y clínicas.

## Qué hay ahora
- Modelo de datos inicial.
- Roles del sistema.
- Reglas simples de recargo por empleado.
- Flujo conversacional del cliente en chat.
- Notificaciones y handoff al empleado.
- Dashboard del dueño con métricas principales.
- Reputación agregada a partir de valoraciones.
- Persistencia local en JSON.
- Tests de dominio, chat, dashboard y storage.

## Siguiente paso
1. Servicio de reservas.
2. Reglas de disponibilidad y agenda.
3. Integraciones de almacenamiento y consulta.
4. Integraciones externas cuando toque.
